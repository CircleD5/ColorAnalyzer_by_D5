# Color Analyzer by D5

イラストを以下のグラフを作って分析します。
- カラーヒストグラム
- 輝度ヒストグラム
- 主要カラー分布
- RGB色空間分布（3D散布図）
- Hue & Saturation 分布（2D散布図）
- ヒートマップ(RBG,彩度, Hue)

以下のURLにアクセスすることでブラウザとネット環境があるなら、どこでも使用することができます。

https://circled5.github.io/ColorAnalyzer_by_D5/
